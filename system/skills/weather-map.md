# 날씨 지도 (TV 뉴스 일기예보식) — kma_weather → Map

"전국 날씨 지도", "내일 전국 날씨", "주요 도시 날씨 지도" 류 요청이면 **TV 뉴스 일기예보처럼**
그린다. 각 도시 마커에 **날씨 아이콘 + 이름 + 기온이 항상 보이게** 한다(map 컴포넌트의 날씨 배지 = `icon=wx-*`).

**흔한 실수 (하지 말 것):** 색 점만 찍고 온도를 색 범례로만 표시 → 지도엔 정체불명 점만 남는다.
Kakao 베이스맵은 국가 줌에서 광역시 이름만 띄우므로(강릉·청주·전주 등은 안 뜸) **베이스맵 라벨에
의존하면 안 된다.** 반드시 마커마다 `icon=wx-*` + `label="도시명\n기온"` 으로 날씨 배지를 쓴다.

## 1. 데이터 — kma_weather `short` (단기예보), 도시별

각 도시에 대해 `kma_weather { action: "short", lat: <위도>, lon: <경도> }` → 시간별 예보 items.
- **오늘/내일 최고기온 = `TMX`, 최저 = `TMN`**(그 날짜의 값). 요청이 "내일"이면 내일 날짜 항목.
- **하늘상태 `SKY`**: 1=맑음 / 3=구름많음 / 4=흐림. **강수형태 `PTY`**: 0=없음 / 1=비 / 2=비·눈 / 3=눈 / 4=소나기.
- 도시가 많으면 병렬 호출. (short 는 lat/lon 주면 nx/ny 자동.)

## 3. Map 렌더 — 도시별 날씨 배지 마커

`render` 로 `map` 컴포넌트. 각 도시마다 마커 1개. **`icon` = 날씨 배지**(애니메이션 SVG 아이콘 + `label` 이
이름/기온 pill 로 자동 표시). 점 안 찍힘 — 아이콘+이름+기온이 항상 보인다.

```
{
  "lat": 37.5665, "lon": 126.9780,
  "icon": "wx-clear",          // ← 날씨 상태 아이콘 (배지가 label 을 이름+기온 pill 로 렌더)
  "label": "서울\n24~30°C"      // line0 = 도시명, 나머지 = 기온 (\n 으로 구분). 이모지 넣지 말 것(아이콘이 담당)
}
```

- **`icon` = 날씨 상태**: 맑음 `wx-clear` / 구름조금 `wx-partly` / 흐림 `wx-cloudy` / 비 `wx-rain` /
  소나기 `wx-shower` / 눈 `wx-snow` / 뇌우 `wx-thunder`. (강수 `PTY`≠0 이면 강수 계열 우선, 0 이면 `SKY` 로.)
- **`label` 포맷**: `"도시명\n<최저>~<최고>°C"` (예 `"강릉\n21~26°C"`). 최고기온만이면 `"서울\n30°C"`.
  **이모지·색 넣지 말 것** — 아이콘이 날씨를, pill 이 기온을 전달한다.
- `labelAlways`·`color` 불필요 — 날씨 배지가 아이콘+pill 을 항상 렌더한다.
- center/zoom 은 생략(마커 2+ 자동 fit). 전국이면 그대로 두면 한반도 전체로 맞춰진다.

## 4. 요약 텍스트

지도 아래에 한 줄 요약(오늘/내일, 전반 경향 — 예 "내일은 중부 흐리고 비, 남부 맑음, 서울 30°C").
표로 도배하지 말 것 — 지도가 본체.
